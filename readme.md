# Arabic Franco 👳‍♂️

convert Arabic words/letters to Franco/English ones. (Franco)

## How to install 🧐

```
npm i arabic-franco
```

## Example Usage

```js
import toFranco from 'arabic-franco';

console.log(toFranco('سارة')); // sara
```
